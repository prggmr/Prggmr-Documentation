Prggmr Event Framework Guide

Welcome to the Prggmr Guide! This guide aims to provide a complete resource for developing event driven software in PHP using the Prggmr Event Framework. The "Hello Events" tutorial dives right into using the Prggmr API, a great place to start learning the code. The "Prggmr API Introduction" explores the Prggmr Event Framework library, and the Event Driven Architecture provides an introduction into the EDA.
